DungenGhost API Documentation

How to start?

1. Create an empty folder in "mods" folder, folder name and will be the name of the mod. Exmaple: "mods\MyNewMod"

2. In the folder of your mod, create a subfolder with name "Sprites", all images that your mod will use will be stored here. Exmaple: "mods\MyNewMod\Sprites"

3. Create an empty xml file in your mod folder. To do this, open any text editor and save the file with the extension .xml and name it "items"

Now you have everything to start add new items in game.

How add new items?

All information about your new items will be stored in the file items.xml and to create new items you must have basic knowledge in the XML markup language.

What is general elements?

General elements these are the elements that should be in each item that you will create. Let's take a look at them.

<type> - This type of item we already added it.
<inv_png> - The name of the picture that will be used for this item in the user interface. (inventory, shop, pickup notifications and etc).
<name> - This name of your item, Id and this name should be same.
<desc> - This just description for your item, this text you can see if click on item in inventory or in shop.
<price> - This how many gold you need to buy this item, or how many gold you will get if you sell this item.
<modname> - Here you need write name of your mod(mod's folder name) so that the game could understand to which mod this item belongs.

Alright! General elements on the spot! Now we can add other elements. Every type of items armor, weapons and etc have own required elements, each type have different. But we in this exmaple create weapon, then we need elements for weapon type.

<png> - This image of weapons in the hands of the protagonist. Images templates you can find on this site in tab "Image templates"
<slot> - This name of slot for equiping, in this case should be "Hands"
<dmg_min> - This is the minimal damage a weapon can del.
<dmg_max> - This is the maximal damage a weapon can del.

Elements can go in any order it doesn't matter.

Good job! If you did everything correctly when you turn on the mod you will be able to find your item in the game.
