DungenGhost API Documentation

General elements:

Elements:

<type> - This type of item, possible types: armor, weapon, material
<inv_png> - The name of the picture that will be used for this item in the user interface. (inventory, shop, pickup notifications and etc).
<name> - This name of your item, Id and this name should be same.
<desc> - This just description for your item, this text you can see if click on item in inventory or in shop.
<price> - This how many gold you need to buy this item, or how many gold you will get if you sell this item.
<modname> - Here you need write name of your mod(mod's folder name) so that the game could understand to which mod this item belongs.
